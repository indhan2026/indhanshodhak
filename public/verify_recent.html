⛽ How to verify a Petrol Pump
1️⃣Open application — View all 3 documents (License, Aadhaar, Selfie at pump)
2️⃣Check: License number matches? Photo clear? Selfie shows person at pump?
3️⃣Click Approve → Pump gets ✅ green tick + unique Pump ID generated
4️⃣Temp password auto-generated → emailed to pump owner for login
5️⃣Aadhaar automatically deleted from server for privacy
❌If documents unclear — click Reject → Enter reason → Email sent to owner
👤 How to verify a User Fuel ID (P1-P4)
1️⃣Check category applied for (P1=Emergency, P2=Doctors, P3=Transport, P4=Farmer)
2️⃣Verify Aadhaar — name and number visible? Vehicle RC — matches application?
3️⃣Check profession certificate — MCI for doctors, permit for truck, KCC for farmer
4️⃣Approve → QR code auto-generated with correct tier → Welcome email sent
5️⃣User's fuel limit upgrades: P5(10L) → P2(20L) → P3(50L) → P4(30L) → P1(Full)
❌Reject with reason → Documents deleted → User notified to resubmit
🚗 Important — Family Member Vehicle RC Rule
A user may have a valid profession (Doctor, Farmer etc.) but the vehicle may be registered in the name of a close family member. This is acceptable and should be approved.
✅Acceptable family members: Spouse (Wife/Husband) · Father · Mother · Son · Daughter · Brother · Sister
✅User uploads: Own Aadhaar + Family member RC + Proof of relation (Ration card / Family certificate) + Profession certificate
✅Example: Dr. Sunita Patil applies as P2. Car RC in husband's name → APPROVE if MCI certificate genuine
✅Example: Farmer Ramesh's tractor RC in son's name → APPROVE with valid KCC and 7/12
⚠️If RC is of a distant relative or non-family person → ASK for clarification → if not satisfied → REJECT
❌DO NOT reject simply because RC name differs — check relation proof first
